# Sales Performance Analytics

A practical business intelligence project focused on analyzing sales data to uncover insights about revenue, customers, products, pipeline health, and team performance.

## Overview

This repository demonstrates how raw sales data can be transformed into actionable business insights using SQL, Python, and Power BI.

## What This Project Covers

- Revenue analysis
- Customer and account performance
- Product performance
- Sales pipeline tracking
- Team and regional contribution

## Dataset

The project uses the following files:

- accounts.csv
- products.csv
- sales_pipeline.csv
- sales_teams.csv

## Project Structure

- sales_database_analysis.sql — SQL queries for analytics
- sales_data_analysis.ipynb — Jupyter Notebook for exploratory analysis
- sales_analytics_dashboard.pbix — Power BI dashboard
- CSV files — source data

## Getting Started

1. Clone the repository
2. Open the project folder
3. Import the CSV files into your preferred tool
4. Run the SQL queries
5. Open the notebook and dashboard for further analysis
